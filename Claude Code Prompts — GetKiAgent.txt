Prompt 1: Projektsetup + Seitenstruktur
/clear

Erstelle ein neues Webprojekt für eine Single-Page-Website (GetKiAgent).

Tech-Stack: Vanilla HTML + CSS + minimal JS. Kein Framework, kein Build-Step. Die Seite muss ohne Node.js deploybar sein (statisches Hosting, z.B. Netlify/Cloudflare Pages).

Struktur:
/getkiagent-website
├── index.html
├── impressum.html
├── datenschutz.html
├── /assets
│   ├── style.css
│   └── script.js
└── /images (leer, Platzhalter)

Design-Vorgaben:
- Farbschema: Dunkel (Dark Background #0f172a oder ähnlich), Akzentfarbe: Electric Blue (#3b82f6) oder Teal
- Typografie: Inter oder System-Fonts (kein Google Fonts Tracking)
- Mobile-first, voll responsiv
- Smooth Scroll zwischen Sektionen
- Keine Animationen außer subtile Hover-States
- Professionell, clean, modern — Referenz-Look: deinekiagentur.com oder ki-agentur.com

Sektionen in index.html (in dieser Reihenfolge):

1. NAVIGATION — Sticky, transparent auf Hero, solid beim Scrollen. Links: Services, Über mich, Preise, Demo, Kontakt. CTA-Button: "Kostenloses Erstgespräch"

   Darunter ein Satz: "Ich baue KI-Systeme, die diese Probleme in 2–4 Wochen lösen."

4. SERVICES — 3 Cards nebeneinander (responsive: untereinander auf Mobile).
   Card 1 "Starter": 1 n8n-Workflow, z.B. automatische E-Mail-Sortierung oder FAQ-Bot. 1–2 Wochen. E-Mail-Support.

Prompt 3: Voiceflow + Loom Embed
/clear

Lies die bestehende Website in /getkiagent-website.

Aufgaben:

1. VOICEFLOW-WIDGET einbauen:
   Füge in der Demo-Sektion dieses Script ein (vor </body>):
   <script type="text/javascript">
     (function(d, t) {
         var v = d.createElement(t), s = d.getElementsByTagName(t)[0];
         v.onload = function() {
           window.voiceflow.chat.load({
             verify: { projectID: 'DEIN_PROJECT_ID' },
             url: 'https://general-runtime.voiceflow.com',
             versionID: 'production'
           });
         }
         v.src = "https://cdn.voiceflow.com/widget/bundle.mjs"; v.type = "text/javascript";
         s.parentNode.insertBefore(v, s);
     })(document, 'script');
   </script>

   Lass DEIN_PROJECT_ID als Platzhalter mit einem HTML-Kommentar <!-- Ersetze DEIN_PROJECT_ID mit deiner Voiceflow Project ID -->.

   Der Chat-Widget von Voiceflow erscheint als Bubble unten rechts — das ist das Default-Verhalten, kein zusätzliches Styling nötig.

2. LOOM-EMBED einbauen:
   In der Demo-Sektion, unter dem Voiceflow-Text, ein responsives Loom-Embed:
   <div style="position: relative; padding-bottom: 56.25%; height: 0;">
     <iframe src="https://www.loom.com/embed/a243a6f8c920487a9db15e9c9816c36e" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"></iframe>
   </div>
   Darüber ein Label: "Sehen Sie unseren KI-Support-Agent in Aktion — 3-Minuten-Demo"

3. Prüfe dass der Voiceflow-Bubble-Button nicht den Calendly-Embed oder andere UI-Elemente überlappt. Falls nötig, padding-bottom auf der Kontakt-Sektion erhöhen.
